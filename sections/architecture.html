FORAP Architecture

FORAP architecture is centered around PjBL principles, design goals, and the support packages that structure reusable and adaptable projects.

Guiding Principles
PjBL Principles
  • Authentic Learning
  • Active Learning
  • Self-Directed Learning
  • Collaborative Learning
  • Critical Thinking
Design Goals
  • Alignment with Learning Objectives
  • Reusability and Adaptability
  • Support for Teaching and Learning
  • Appropriateness of Challenges
  • Formative and Summative Assessment
Support Packages

  • Sample Solutions
  • Instructor Notes
  • Reference Course
Student Support Package
  • Installation Guidelines

  • Laboratory Exercises
  • Reference Materials
Assessment Package

  • Transfer-oriented Design Problems
